Apache Webserver
2023

Fully self-hosted and maintained Apache webserver with valid and current personal SSL/TLS certificates. Uses personally owned Raspberry Pi 3 B+ hardware running a fork of Debian with custom DNS name servers to maintain 99% uptime. Used knowledge of HTLM, CSS, and JavaScript to design and create a website by hand without the use of any external tools. The webserver is hosting the very webpage you are viewing right now.

What is Donau?

Donau is the personal website of Lukas Krampitz. It is the home of many projects. One such project is the webserver that is hosting this very page. Another project was designing and creating the front-end of the static website too. Donau.ca also showcases quite a few other personal projects too. One such project is the restoration, cleaning, and modification of IBM Model M and M122 keyboards. Donau.ca is also home to many other miscellanies things such as travel pictures. The light house on the left is found on the German island of Amrum.
